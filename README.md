Une application météo simple construite avec Laravel.

## Installation

1. **Cloner le projet**
   ```bash
   git clone https://github.com/M-Hnida/appmeteo.git
   cd appmeteo
   ```

2. **Installer les dépendances PHP**
   ```bash
   composer install
   ```

3. **Installer les dépendances Frontend**
   ```bash
   pnpm install
   ```

4. **Lancer l'application**
   ```bash
   php artisan serve
   ```

Accédez à l'application via : `http://127.0.0.1:8000/weather`

## Technologies utilisées

 Laravel 11

 PHP 8.2+

 Blade & Tailwind CSS

 [Open-Meteo](https://open-meteo.com/)
